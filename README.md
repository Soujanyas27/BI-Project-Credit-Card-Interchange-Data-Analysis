# BI-Project-Credit-Card-Interchange-Data-Analysis
Business Intelligence Project analyzing credit card interchange transaction data using SQL and Power BI to generate business insights.

PROJECT OVERVIEW: 
The objective is to simulate real-world BI tasks such as interpreting business requirements, designing data structures, writing SQL queries, and creating analytical reports.
The dataset contains three months of credit card interchange transaction data provided in CSV format. Using this data, the project demonstrates how raw transactional data can be transformed into meaningful insights for business decision-making.

# Objectives
The primary goals of this project are:
- Interpret business reporting requirements
- Design appropriate database tables
- Import and transform raw CSV data
- Write SQL queries to analyze transaction data
- Build analytical reports and dashboards
- Present insights derived from the dataset

# Technologies Used
- SQL : Data transformation and querying
- Microsoft SQL Server / SSMS : Database creation and query execution
- Power BI : Data visualization and dashboard creation
- CSV Data Processing : Data ingestion and cleaning

# Key Skills Demonstrated
- Data Engineering fundamentals
- SQL querying and optimization
- Data modeling
- Business Intelligence reporting
- Data visualization
- Analytical problem solving
